# Project Description :-
- LearnTrack is a console-based Student & Course Management System built using Core Java.

# Steps to compile and run :- 


## compile (Main)
	javac -d out -sourcepath src/main/java src/main/java/com/airtribe/learntrack/Main.java
## Run
	java -cp out com.airtribe.learntrack.Main
	
	
## compile (Console UI Main)
	javac -d out -sourcepath src/main/java src/main/java/com/airtribe/learntrack/ui/Main.java
## Run
	java -cp out com.airtribe.learntrack.ui.Main
